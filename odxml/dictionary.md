# &lt;dictionary&gt;

Dictionary nodes occur at the base of all source files and will not compile without one. ODict looks for these nodes by default when compiling. They take no attributes.

## Attributes

| Name | Type | Required? |
| :--- | :--- | :--- |
| name | String | No |

## Children

| Name | Required? |
| :--- | :--- |
| [&lt;entry&gt;](entry.md) | Recommended |
